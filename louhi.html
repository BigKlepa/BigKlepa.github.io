Louhi

Louhi began as a student project, where I was the lead programmer, spokesperson, and project manager. In Louhi, you play as a forest warden, investigating disturbing and mysterious events in a late 1800's village. It was a single player top-down action-adventure, inspired by Zelda: A Link To The Past, Dark Souls, HP Lovecraft, and Kalevala

The project is no longer in development, but you can see an old repository here.

I'm happy to say that my programming habits have improved a lot since then.

Louhi was created with Unity, utilizing its then fairly new tileset feature, and it's old clunky input manager.

I learned a lot about project management, long-term production, and entrepeneurship. We entered the Boost Startup Journey with our game, and got a lot of positive feedback.

Tools and Technologies: Unity, C#, Visual Studio, Aseprite, GIMP, Trello, GitHub
